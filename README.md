
# ElCoffeino

![presentation](https://raw.githubusercontent.com/KamilDziuk/ElCoffeino/main/public/gif/presentation.gif)

A modern one-page website for **ElCoffeino**, combining a premium mobile coffee bar experience with freelance chef services.

## Project Overview

This project is a sleek, high-performance one-page website designed to present services in a clean, engaging, and modern way.

The website focuses on:
- showcasing a mobile coffee bar
- promoting freelance chef services
- delivering modern UI/UX design
- ensuring high performance and fast load times

## Tech Stack

Built using modern frontend technologies:

- **React** – UI library
- **TypeScript** – static typing
- **Vite** – fast development/build tool
- **Tailwind CSS** – utility-first styling

## Features

-  Fast performance powered by Vite
-  Fully responsive (mobile-first)
-  One-page layout
-  Modern UI/UX design
-  Optimized for performance
-  SEO-friendly structure

## Installation & Setup

 Clone the repository:
```bash
git clone https://github.com/your-repo/elcoffeino.git
```
Navigate to the project directory:
```bash
cd elcoffeino
```
Install dependencies:
```bash
npm install
```
Start development server:
```bash
npm run dev
```
Production Build:
```bash
npm run build
```
